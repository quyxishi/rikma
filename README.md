Usage
----
* Encrypt
```
$ python rikma.py -e
```
* Decrypt
```
$ python rikma.py -d
```
* Encrypt/decrypt all folders in your path
```
$ python rikma.py -e -a
$ python rikma.py -d -a
```
* Get help
```
$ python rikma.py -h
```
