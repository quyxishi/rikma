Usage
----
* Encrypt
```
$ python rikma.py -e
```
* Decrypt
```
$ python rikma.py -d
```
* Encrypt all folders in your path
```
$ python rikma.py -e -a
```
* Get help
```
$ python rikma.py -h
```
