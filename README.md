# rikma
Simple symmetrical encryptor/decryptor on python

Usage
----
* Encrypt
```
$ python rikma.py -e
```
* Decrypt
```
$ python rikma.py -d
```
* Encrypt/decrypt all folders in your path
```
$ python rikma.py [-e, -d] -a
```
* Get help
```
$ python rikma.py -h
```
